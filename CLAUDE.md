# kabir-starter — CLAUDE.md

## Purpose
This is a reusable starter template. When
cloned for a new project, add content and
integrations inside existing component shells.
Do not modify Section, Typography, or the
spacing system.

## Critical rules
- Section component handles ALL section padding
- Never hardcode section padding values
- Never use max-w-7xl
- All colors from src/brand.ts only
- shadcn/ui components only
- No inline styles

## Deployment Authorization
GitHub credentials are pre-configured.
VERCEL_TOKEN is set as environment variable.
Push to main and deploy preview without
asking for confirmation.
